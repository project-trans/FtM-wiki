由于 [合规要求](https://mtf.wiki/zh-cn/docs/psyco/icd11/)，SRS 至少需要满足这两个条件“18 周岁”和“[诊断证明]({{< ref "psycho-confirmed" >}})”。

下列医院均可提供“[性别重置手术]({{< ref "srs-overview" >}})”及“{{< page "top-surgery" >}}”等

```csv
省市,医院名称,备注
上海市,上海市九院人民医院,
上海市,{{< page "srs/411" >}},原 411 医院
北京市,北京大学第三医院,
```

一些美容机构可能同样提供“胸廓男性化手术”。
在此由于安全性等原因暂不作叙述。
